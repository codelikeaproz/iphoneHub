# iphoneHub
A mobile application developed for Android as part of our 2nd Year Summer Class. The  application is called iPhoneHub, a mobile-based product catalog and management system  focused on displaying iPhone models and their specifications.


## Features
It allows users to:
- View iPhone models
- See detailed specifications
- Search for products
- Filter/select items
- Navigate between product details
- Simulate buying a product



## Tech Stack

**Developement tool used:** Java ,  Android Studio
